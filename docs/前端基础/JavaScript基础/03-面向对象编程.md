### JS中描述对象
> 可以使用```Object.defineProperty(obj,props,description)```进行定义
1. 数据属性
- configurable
- enumerable
- writable
- valuef
```js
let person = {}
Object.defineProperty(person,'name',{
    writable:false,
    value:'bob'
})
```
2. 访问器属性
- get
> 访问属性的时候，会默认调用这个get函数
- set
- configurable
- enumerable
