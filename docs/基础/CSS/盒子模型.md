### 盒模型的分类

css的属性display来设置

- border-box
- content-box (默认)

**两者计算宽高的方式不一样，前者将border计算在内，后者不计算**使用`contetn-box`省事



### JS获取盒模型的宽高

- `dom.style.width/height`
- `dom.currentstyle.width/height`
- `window.getComputedStyle(dom).width/height`
- `dom.getBoundClientRect().width/height`
