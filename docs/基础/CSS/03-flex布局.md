### flex
flext用来生成flex容器，是一种很好的布局方案，对移动端适配性很好。

### flex的属性 (6个)
- flex-direction
    - row
    - row-reverse
    - column
    - column-reverse
    
- **flex-wrap**
    
    - nowrap
    - wrap
    - wrap-reverse
    
- **flex-flow**

- **justify-content**
    
    - flex-start
    - center
    - flex-end
    - space-between
    - space-around
    
    ![image-20211201000952879](./img/image-20211201000952879.png)
    
- align-items
    - flex-start
    - center
    - flex-end
    
- align-content
